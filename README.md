## Run Locally

Clone the project

```bash
  git clone https://github.com/DevShaunB/fem-single-price-grid-component.git
```

Go to the project directory

```bash
  cd fem-single-price-grid-component
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
